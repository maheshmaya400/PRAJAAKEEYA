# 🤝 Contributing to PRAJAAKEEYA | ಪ್ರಜಾಕೀಯ

First off, thank you for considering contributing! This project is a **community-driven,
politically neutral, educational resource**, and every contribution — big or small — helps
make civic education more accessible.

---

## 📋 Before You Start

Please read:
- [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)
- The **Core Rules** in [README.md](README.md): educational only, politically neutral,
  no propaganda, no misinformation, beginner friendly, research-based.

All contributions must remain strictly **neutral and factual**. Content that promotes,
attacks, or endorses any political party, individual, or ideology will not be accepted.

---

## 🧑‍💻 How Developers Can Help

- Improve Markdown structure and accessibility
- Build/maintain Mermaid diagrams
- Add automated link-checking or markdown-lint workflows
- Improve mobile rendering of tables and diagrams

## 🎨 How Designers Can Help

- Create or refine assets in `assets/banner/` and `assets/images/`
- Ensure all visuals are modern, mobile-friendly, and politically neutral
- Maintain consistent color palette and typography across assets

## ✍️ How Writers Can Help

- Improve clarity and tone of existing English/Kannada content
- Add well-sourced academic context
- Expand the FAQ with beginner-friendly explanations

## 🌐 How Translators Can Help

- Improve Kannada translation accuracy
- Add additional regional languages (Hindi, Tamil, Telugu, Marathi, etc.)
- Ensure cultural and linguistic nuance is preserved

## 🔬 How Researchers Can Help

- Verify and cite claims with credible, neutral sources
- Flag any content that may read as biased or unverified
- Suggest academic references on governance theory and civic participation

---

## 🔁 Contribution Workflow

1. **Fork** this repository
2. **Create a branch**: `git checkout -b feature/your-feature-name`
3. **Make your changes** following the style guide below
4. **Commit**: `git commit -m "docs: improve Kannada translation in FAQ"`
5. **Push**: `git push origin feature/your-feature-name`
6. **Open a Pull Request** with a clear description of your changes

---

## 🖋️ Style Guide

- Use clear, simple language (assume a beginner reader)
- Maintain bilingual parity — if you edit English content, update the Kannada
  equivalent (or flag it for translation)
- Use proper Markdown headings (`##`, `###`) — don't skip levels
- Keep tables and diagrams mobile-responsive (avoid overly wide tables)
- Cite sources for any factual/historical claims where possible
- Avoid emotionally charged or persuasive language — stay descriptive, not prescriptive

---

## ✅ Pull Request Checklist

- [ ] Content is neutral and factual
- [ ] No political endorsements or attacks
- [ ] Bilingual content is consistent (English + Kannada)
- [ ] Markdown renders correctly (preview before submitting)
- [ ] Image paths match the exact folder structure (`assets/banner/`, `assets/images/`)
- [ ] No broken links
